# Image to ASCII Converter

> A Python 3.13.4 application that converts images into ASCII art.

---

## Dependencies
`Pillow:` To install it, open your Command Prompt (Windows) or Terminal (macOS/Linux) and run the following command:

```bash
pip install Pillow
```
